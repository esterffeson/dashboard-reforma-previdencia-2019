## Instruções de uso do dashboard.

### Foi utilizado a ferramenta Qlik Sense Desktop

* Salve o arquivo Previdência.qvf no caminho Documents\Qlik\Sense\Apps (Por padrão é nesse caminho que o Qlik Sense armazena seus relátorios)
* Ao abrir o Qlik Sense é preciso alterar o caminho de conexão dos dados. Com o relatório Previdência aberto vai em:
  1. Editor de carga de dados
  1. No carto direito -> Selecionar Dados
  1. Procure a planilha do repositório Previdência.xlsx e dê ok.
  1. Selecione todas as colunas e clique em "Inserir Scripts".
  1. Clique em Carregar Dados.
